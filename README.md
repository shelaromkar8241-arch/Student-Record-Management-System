# Student Record Management System

## Overview
A Command Line Interface (CLI) application built in Python to manage student data. It allows users to perform basic CRUD (Create, Read, Update, Delete) operations on student records. All data is persistently saved to a JSON file.

## Features
* **Add Record**: Enter a student's roll number, name, age, marks, and subjects.
* **View Records**: Display all stored student details.
* **Update Record**: Modify existing student information.
* **Delete Record**: Remove a student from the database.
* **Data Persistence**: Uses `students.json` for file handling so data is saved between sessions.

## Technologies Used
* Python 3
* JSON module (Standard Library)
